# 🚀 Personal Portfolio Website

This is a modern, responsive personal portfolio website built with **Next.js, React, Tailwind CSS, and Framer Motion**.  
It showcases my skills, projects, and provides a way to contact me.

## ✨ Features

- ⚡ Modern UI with smooth animations (Framer Motion)
- 🌙 Dark / Light mode support
- 📱 Fully responsive (mobile, tablet, desktop)
- 🎯 Skills section with tech stack icons
- 📬 Contact form using FormSubmit
- 🎨 Clean and minimal design
- 🚀 Fast performance with Next.js

---

## 🛠️ Tech Stack

**Frontend:**
- Next.js
- React.js
- Tailwind CSS
- Framer Motion

**Tools & Services:**
- FormSubmit (for contact form)
- Git & GitHub
- Vercel (deployment)
